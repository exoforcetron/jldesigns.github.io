Collage Effect animation

Description

For this class exercise we were to make a collage style animation. 
I was listening to music and picked Matryoshka to be the basis for the animation. Matryoshka is a silly song that is more vague with a upbeat over sad and crazy feeling. 
I do not own the rights to the song Matryoshka.

For each image I would select the image/text with lasso tool then add layer mask. Put a paper grain and torn paper image behind it. Blend the torn paper. Create a path from the layer mask selection and use a jagged brush. This creates the torn paper effect.

For the text portion I made each letter into a different font and to make it look all random. The text was separated into each letter. Then the same torn paper process of cutting out each letter.After that each letter was saved then imported into after effects animated individually. Each image was rotated and shaken to add unease. Below is just one line of text that is "aa, shiritai na". For a total of 21 lines of text.

For the exploding head was made using 38 individual images to create the effect. I chopped up the image and added torn effect and pieced it back together.
